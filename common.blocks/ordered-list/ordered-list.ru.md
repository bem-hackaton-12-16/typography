# ordered-list

Блок предоставляет компонент интерфейса для отображения нумерованного списка.

## Пример использования

```js
{
    block: 'ordered-list',
    content: [
        { elem: 'item', content: 'Milk' },
        { elem: 'item', content: 'Coffee' },
        { elem: 'item', content: 'Apple' }
    ]
}
```
